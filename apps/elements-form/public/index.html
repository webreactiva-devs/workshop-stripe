Stripe Embedded Forms Demo

Proceso de pago con Stripe

Esta es una demostración de integración de formularios de pago embebidos de Stripe.

Sobre esta demo

Esta aplicación demuestra cómo integrar el sistema de pago de Stripe usando formularios embebidos directamente en tu sitio web, proporcionando una experiencia de pago sin redirecciones.

También incluye una implementación de webhooks para procesar eventos de Stripe.
